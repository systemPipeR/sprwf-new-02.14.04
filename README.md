## SystemPipeRdata: Generic Workflow Template

This is the Generic Workflow Template for systemPipeRdata.

__Usage__

Consult the details [here](https://systempipe.org/sprwf-new-02-14-04/new.html) or click the workflow's HTML filename in the repository.

To download this workflow, run one of the following commands. The `download_link` can be obtained under the green Code button on the top right of the respository. Usually,
the HTTPS option is the easiest for most users. 

From the command-line:

```
git clone <download_link>
```

From R:

```
gert::git_clone(<download_link>)
```
